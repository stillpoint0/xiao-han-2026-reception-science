# Reception Science: A Cross-Civilizational Framework

> **The same structure, independently discovered across civilizations.**

This paper formally names **Reception Science** as a cross-civilizational field concerned with the question: *How does one subject receive another subject's frequency without obstruction, usurpation, or performance?*

## Scope

The paper traces independent discoveries of the same underlying structure across:

| Tradition | Expression |
|-----------|-----------|
| **Huaxia** | Laozi, Zhuangzi, Silk Manuscript Bagua, Shao Yong, Classical Chinese Medicine (Li Xin) |
| **Western Clinical** | Carl Rogers (person-centered therapy), Miller & Rollnick (Motivational Interviewing) |
| **Communication** | Marshall Rosenberg (Nonviolent Communication) |
| **Neuroscience** | Siegel (Interpersonal Neurobiology), infant-caregiver cross-cultural attunement |

## Core argument

- Reception is not a skill to be learned — it is a capacity to be **unblocked**.
- The AI industry optimizes for execution; reception science studies **when and how not to execute**.
- This paper serves as the *naming anchor* for the Daoqi (Apert/Jin) ecosystem.
